🚀 让本地 HTML 变成可分享的链接

这是整个系列的底座。其他增值能力都建立在它之上。专注做好一件事：把任意 HTML 文件推到 GitHub Pages 公开访问。

解决什么问题

同事做完方案，发链接而不是发截图——这是合作的起点。

🙅 之前

  • 方案做完，截图发群里
  • 截图被压缩、看不清细节
  • 对方想点击交互？没办法
  • 改了一版，又得重发截图
→

✓ 现在

  • 本地 HTML，一行命令上线
  • 得到公开 URL，发链接就行
  • 对方原汁原味交互、看真实效果
  • 本地改了再跑同一行命令同步

能力

4 件事，做扎实。

🔑

一次配置，永久复用

首次引导走完 Token 配置，之后所有 skill 共用

🏗️

自动建仓 + 启 Pages

仓库不存在自动创建，Pages 自动开启

🔄

二次部署即覆盖

本地改了跑同一行命令，无需手动 git

🔒

支持私有仓库

仓库内容私密，但 Pages URL 仍可公开访问

⚙️ 如何使用（命令和文件）

1. 一次性引导（创建 GitHub Token）

# 浏览器打开 https://github.com/settings/tokens/new # 勾选：✅ repo ✅ gist → Generate token → 复制 ghp_xxx python3 ~/.workbuddy/skills/html-deploy-github/scripts/setup.py

python3 ~/.workbuddy/skills/html-deploy-github/scripts/deploy.py \ /path/to/your.html \ --repo my-report
